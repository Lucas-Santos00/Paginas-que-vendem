FAQ

  • Posso pedir alterações depois?

    Sim! Incluímos pequenas alterações para ajustes visuais ou textuais. Caso precise de mudanças maiores, combinamos novos valores antes.

  • Funciona no celular?

    Com certeza. Todas as páginas são responsivas, adaptadas para celulares, tablets e desktops, garantindo a melhor experiência em qualquer tela.

  • Qual o prazo de entrega?

    Depende do tamanho e complexidade do projeto. Após entender sua necessidade, envio um orçamento com prazo realista.

  • WordPress vs Código

    WordPress é rápido para sites simples, mas páginas feitas com código (HTML, CSS, JS e até React/NextJS) oferecem mais desempenho, personalização e segurança.

  • Quais tecnologias são usadas?

    A base é HTML, CSS e JavaScript, mas quando necessário utilizo React, NextJS e outras tecnologias para entregar performance e escalabilidade.
